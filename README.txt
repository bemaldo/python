-----------------------
Repository Description
-----------------------
The python repository contains modules to be used for general purposes.  These
are designed with a focus on practicing the python programming language.

-----------------------
Repository structure
-----------------------
trunk - contains the modules to work on.  checkout in a branch under branches/branchname
	to work without modifying the drunk until changes are merged/commited into the 
	trunk at the appropriate time.
branches - contains branches created by user. each branch is to contain committ messages 
	describing their creation purposes, issue, etc.
